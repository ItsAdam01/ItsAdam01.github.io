# Adam Atienza - Cybersecurity Specialist Portfolio

A professional portfolio and blog showcasing cybersecurity research, secure software development, and specialized security hardening projects.

## Projects

- **Sentinel Vault**: Identity & Access Management proof-of-concept.
- **NIDS & WAF**: Real-time network security monitoring system.
- **Lynx**: High-performance File Integrity Monitor built in Go.
- **Blueship Hardening**: Security overhaul for a commercial resort booking platform.

## Built With

- **Astro**: Static site generation.
- **Tailwind CSS**: Styling.
- **TypeScript**: Type-safe development.
- **SolidJS**: Interactive components.

## Development

All commands are run from the root of the project:

| Command | Action |
| :--- | :--- |
| `pnpm install` | Installs dependencies |
| `pnpm dev` | Starts local dev server |
| `pnpm build` | Build for production |
| `pnpm preview` | Preview local build |
